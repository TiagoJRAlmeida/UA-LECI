Erro de unicode object: Usar ".encode()" para tranformar em bytes e resolver problema.
Source: The unicode object lets you work with characters. It has all the 
same methods as the string object. “encoding” is converting from a unicode 
object to bytes. “decoding” is converting from bytes to a unicode object.

Ao enviar dados que estão em formato bytes tem de se fazer:
    - base64.b64encode(data).decode()

Para receber os dados e transformar em bytes denovo para usar basta fazer:
    - base64.b64decode(data)


O cliente ao receber algo do repositorio, ao usar o metodo .json(), por exemplo:
    data = response.json()

"data" já está no formato dict, logo não é necessário fazer json.loads()
